# Design System

This folder describes **how ORA Navigator is built and what it is built with**. It is the
"big picture" — read it before diving into any single feature.

| File | What it answers |
|---|---|
| [`architecture.md`](architecture.md) | How is the system structured? What are the services, how do they talk, how does an answer get produced, how is it deployed? |
| [`tech-stack.md`](tech-stack.md) | What is it made of? Languages, frameworks, UI libraries, cloud services, dev & test tooling. |
| [`agents.md`](agents.md) | How should all the AI agents work? The shared contract (AI proposes, code verifies, human confirms), the principles, and a brief on each agent. |

For *what* the product does (feature by feature), see [`../features/`](../features/).
